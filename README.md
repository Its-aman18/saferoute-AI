SafeRoute AI is an AI-powered road safety platform that predicts accident-prone zones and alerts users before they enter high-risk areas.

## Problem
Road accidents occur due to lack of predictive safety systems and real-time risk awareness.

## Solution
SafeRoute AI uses Google Maps, BigQuery, and Vertex AI to identify accident hotspots and provide safer navigation insights.

## Google Technologies Used
- Google Maps JavaScript API
- Firebase Hosting
- BigQuery
- Vertex AI
